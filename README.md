# nordic-feed
RSS feeds agregator aka planet for Nordic blogs in Polish.

## Setup

```
virtualenv -ppython3 env
. env/bin/activate
pip install -r requirements.txt
```

* [Pelican docs](https://docs.getpelican.com/en/3.7.1/install.html)

## Updating

```
make
```
